# 🚀 Personal Portfolio Website - Fatin

Website portofolio profesional satu halaman yang dibangun dengan **HTML5**, **Tailwind CSS**, dan **Vanilla JavaScript**. Website ini dirancang untuk menampilkan profil, keahlian, dan proyek utama sebagai mahasiswa Sistem dan Teknologi Informasi STEI ITB.

## 🌟 Fitur Utama
- **Desain Modern & Responsif**: Tampilan estetik yang menyesuaikan dengan perangkat mobile, tablet, dan desktop.
- **Glassmorphism UI**: Menggunakan efek transparansi modern pada elemen-elemen tertentu.
- **Scroll Animations**: Efek *fade-in* saat elemen masuk ke dalam viewport menggunakan Intersection Observer API.
- **Smooth Navigation**: Navigasi halus antar section dengan perhitungan offset navbar otomatis.
- **Sticky Navbar**: Header yang berubah menjadi solid dan mengecil saat di-scroll.
- **Konten Statis**: Pengisian konten langsung (tanpa lorem ipsum) sesuai profil akademik dan organisasi.

## 🛠️ Teknologi yang Digunakan
- **HTML5**: Struktur web semantik.
- **Tailwind CSS (via CDN)**: Framework CSS utility-first untuk styling cepat dan modern.
- **Vanilla JavaScript**: Logika interaktivitas, animasi, dan manipulasi DOM ringan.
- **Font Awesome**: Library ikon untuk visualisasi yang lebih baik.
- **Google Fonts (Inter)**: Tipografi yang bersih dan profesional.

## 📁 Struktur Proyek
```text
.
├── index.html    # Struktur utama dan styling Tailwind
├── script.js     # Logika interaksi dan animasi

```

## 🚀 Cara Menjalankan Secara Lokal
1. **Clone repositori ini:**
   ```bash
   git clone https://github.com/username/portofolio.git
   ```
2. **Masuk ke direktori proyek:**
   ```bash
   cd portofolio
   ```
3. **Buka file `index.html`:**
   Cukup klik dua kali pada file `index.html` atau gunakan extension **Live Server** di VS Code untuk pengalaman pengembangan yang lebih baik.

## 🌐 Deployment
Website ini sangat ringan dan siap di-deploy ke berbagai platform:
- **GitHub Pages**: Masuk ke Settings > Pages > Pilih branch `main` > Save.
- **Vercel**: Hubungkan akun GitHub Anda dan pilih repositori ini untuk deploy otomatis.
- **Netlify**: Drag & drop folder proyek ke dashboard Netlify.

## 📝 Catatan untuk Penyesuaian
- **Foto Profil**: Ganti placeholder ikon pada `index.html` baris hero section dengan tag `<img>` yang merujuk ke file foto Anda.
- **Link Sosial Media**: Pastikan untuk mengubah placeholder link GitHub dan LinkedIn di bagian `Footer` dan `Contact`.
